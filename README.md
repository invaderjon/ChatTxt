# ChatTxt

Dependencies
==========
- Boost System Library 
- Boost Thread Library

Compiling Instructions
==========
-Mac 
	-run 'brew install boost' 
	-run 'make mac'
	Note: if there are compiler errors, check that appended '-mt' to the two boost libraries.
